# SOS Semantic Cloud MVP v0.1.1

**Word Cloud Drift Simulator** — интерактивная 2D-симуляция дрейфа смыслов под влиянием контекста.

```
Word → MeaningClouds → ContextWind → Drift → DominantMeaning → Precipitation(осадки)
```

## ✨ Возможности

### Основной функционал
- ✅ **Ввод слов:** добавление/удаление слов через UI или программно
- ✅ **Смысловые облака:** каждое слово распадается на 3–5 облаков значений
- ✅ **Контекстный ветер:** физическое поле смещает, усиливает или ослабляет облака
- ✅ **Дрейф смысла:** облака перемещаются в 2D пространстве
- ✅ **Осадки смысла:** фиксация интерпретации при достаточной определённости
- ✅ **Системные метрики:** энтропия, вес, кол-во событий осадков

### Новое в v0.1.1
- 🆕 **SaveManager** — сохранение/загрузка состояний симуляции
- 🆕 **TestRunner** — юнит-тесты всех компонентов
- 🆕 **DemoRunner** — автоматическая демонстрация возможностей
- 🆕 **Расширенная UI** — показ доминирующих смыслов в реальном времени
- 🆕 **Системный лог** — детальное логирование всех событий
- 🆕 **Export API** — экспорт состояния в JSON для анализа

## 🛠️ Технологии

- Godot 4.3+
- GDScript 2.0 (strict typing)
- CanvasItem-шейдер для рендеринга
- JSON-словарь (10 слов, 40 значений)

## 📦 Установка и запуск

### Клонирование

```bash
git clone <repo-url>
cd sos_semantic_cloud_mvp
```

### Открытие в Godot

1. Откройте Godot Editor
2. Импортируйте `project.godot`
3. Нажмите F5 для запуска

### Через CLI

```bash
godot .
```

## 🚀 Быстрый старт

1. **Демонстрация:** Проект автоматически запускает `DemoRunner` с полной демо-сессией
2. **Тестирование:** Добавьте `TestRunner` в Main.tscn для запуска юнит-тестов
3. **Интерактивный режим:** Используйте UI для добавления слов и установки контекста

## 📝 Основные операции

```gdscript
# Добавить слово
Simulation.add_word("свет")

# Установить контекст
ContextField.configure(Vector2.RIGHT, 0.6, ["наука", "физика"])

# Получить метрики
var metrics = Simulation.system_metrics()
print("Слов: %d" % metrics["word_count"])

# Сохранить состояние
SaveManager.save_state("my_session")

# Загрузить состояние
SaveManager.load_state("simulation_my_session.json")

# Подписаться на события
Simulation.precipitation_detected.connect(func(word, meaning, weight, entropy):
    print("Осадок: %s → %s" % [word, meaning])
)
```
