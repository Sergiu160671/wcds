## [0.1.1] — 2026-05-28

### ✨ Добавлено

#### SaveManager (новый автозагруженный синглтон)
- `save_state(name: String)` — сохранение текущего состояния в `user://sos_saves/`
- `load_state(filename: String)` — загрузка сохранённого состояния
- `list_saves() -> Array` — получение списка всех сохранений
- `export_json(filename: String)` — экспорт состояния в JSON

#### TestRunner (новый компонент)
- Юнит-тесты для всех основных компонентов
- Проверка загрузки словаря
- Тестирование создания узлов и облаков
- Валидация физики дрейфа
- Проверка работы контекстного поля
- Полный тест симуляции

#### DemoRunner (новый компонент)
- Автоматическая 10-этапная демонстрация
- Добавление слов
- Смена контекста
- Визуализация статистики
- Сохранение результатов

#### Расширенная UI (ui_controller.gd)
- Показ доминирующих смыслов в реальном времени
- Обновлённые метрики с дополнительными данными
- Улучшенная обработка ошибок ввода
- Подключение Enter для добавления слова
- Информативные debug-сообщения

#### Simulation (расширения)
- `frame_count` — счётчик кадров симуляции
- `precipitation_log` — лог всех событий осадков смысла
- `signal precipitation_detected(word, meaning, weight, entropy)` — новый сигнал
- `export_state() -> Dictionary` — экспорт полного состояния
- Расширенные метрики: `avg_weight`, `frame`, `precipitation_count`
- Детальное логирование всех операций

#### Документация
- **[USAGE.md](docs/USAGE.md)** — полное руководство пользователя (7.7 КБ)
  - Примеры использования
  - API документация
  - Типы облаков и колоризация
  - Параметры симуляции
  - Тестирование и отладка
  - Расширение функциональности

## [0.1.0] — 2026-05-25

### ✨ Начальный MVP

- Ядро симуляции с физикой дрейфа облаков
- Контекстный ветер с фильтрацией по ключевым словам
- Детекция доминирующего смысла
- Осадки смысла (precipitation) при достаточной стабильности
- JSON-словарь с 10 словами и 40 значениями
- 6 типов облаков с колоризацией (abstract, practical, conflict, metaphor, technical, neutral)
- Базовый UI для управления симуляцией
- Шейдер для визуализации облаков
- Основная документация (codebase-guide, data-layer, conventions)

### Компоненты

- `Simulation` — главный цикл и управление
- `ContextField` — контекстный ветер
- `SemanticDictionary` — загрузчик словаря
- `WordNode` (RefCounted) — модель слова
- `MeaningCloud` (RefCounted) — модель облака
- `WordNodeView` — визуализация слова
- `MeaningCloudView` — визуализация облака
- `UIController` — управление интерфейсом

### Статистика

```
Общие метрики (v0.1.0):
  - 508 строк GDScript
  - 4 сцены (.tscn)
  - 1 шейдер (cloud_shader.gdshader)
  - 7.8 КБ данных (meanings_ru.json)
  - 5 документов (README, codebase-guide, data-layer, conventions, CLAUDE)
  - Словарь: 10 слов, 40 значений
```

## 📋 Соглашения о версионировании

Проект использует [Semantic Versioning](https://semver.org/):
- **MAJOR** — критические изменения API
- **MINOR** — новые возможности, обратная совместимость
- **PATCH** — исправления ошибок

## 📖 Формат записей

Каждая запись содержит:
- **Добавлено** (`✨ Added`) — новая функциональность
- **Улучшено** (`🔧 Changed`) — изменения существующей функциональности
- **Исправлено** (`🐛 Fixed`) — исправления ошибок
- **Удалено** (`❌ Removed`) — удалённая функциональность
- **Безопасность** (`🔒 Security`) — исправления уязвимостей
